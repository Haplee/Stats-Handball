# Stats-Handball - Plataforma de Análisis de Vídeo con IA

**Stats-Handball** es una aplicación web diseñada para analizar partidos de balonmano y extraer estadísticas detalladas de forma automática. Utilizando técnicas de Visión por Computador e Inteligencia Artificial, la plataforma procesa vídeos de partidos para identificar jugadores, seguir sus movimientos y registrar acciones clave como goles y lanzamientos.

## 🚀 Estado del Proyecto

Este proyecto se encuentra en **desarrollo activo**. Hemos sentado las bases de la aplicación y estamos trabajando continuamente para mejorarla. Puedes seguir nuestro progreso en el [Dashboard de Progreso](frontend/src/views/dashboard.html).

## ✨ Características Actuales

-   **Carga de Vídeos**: Sube un vídeo de un partido para su análisis.
-   **Detección de Jugadores**: Identifica a los jugadores en el campo.
-   **Estadísticas Básicas**: Genera un recuento de goles y lanzamientos.
-   **API REST**: Consulta las estadísticas a través de una API.

## 🛠️ Tecnologías Utilizadas

-   **Backend**: Python, Flask, SQLAlchemy
-   **Frontend**: HTML, CSS, JavaScript
-   **Base de Datos**: SQLite (desarrollo), PostgreSQL (producción)
-   **IA**: PyTorch, YOLOv8, OpenCV
