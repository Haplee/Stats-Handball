# Handball Stats - Progress Web

This is the official progress website for the Handball Stats TFG project. It is a modern Single Page Application built with Vite, React, and Tailwind CSS.

## Development

To run this project locally, follow these steps:

1.  **Navigate to the project directory:**
    ```bash
    cd progress-web
    ```

2.  **Install dependencies:**
    ```bash
    pnpm install
    ```

3.  **Start the development server:**
    ```bash
    pnpm dev
    ```

The application will be available at `http://localhost:5173`.
